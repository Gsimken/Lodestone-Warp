# Installation

**Last updated:** 2026-06-27

## Server Requirements

- Minecraft `26.2`
- Java `25+`
- Fabric Loader `0.19.3+`
- Fabric API
- Fabric Permissions API

## Optional

- LuckPerms for permission management.
- Lodestone Warps installed on the client for the enhanced custom UI.

## Server-Only Setup

1. Install Fabric Loader on the server.
2. Add Fabric API.
3. Add Fabric Permissions API.
4. Add `lodestone-warps-<version>.jar`.
5. Start the server once to generate the config.
6. Edit `config/lodestone_teleport.json` if needed.

Vanilla players can join without installing the mod.

## Client Mod Setup

Players may also install Lodestone Warps client-side.

This enables the custom mod UI with:

- table layout
- search
- pagination
- item cost icons
- edit buttons

Players without the client mod continue using the vanilla Dialog UI.
