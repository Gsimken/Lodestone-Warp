# Installation

[English](Installation.md) | [Español](es/Instalacion.md)

**Last updated:** 2026-07-13

## Server Requirements

- Minecraft `26.2`
- Java `25+`
- Fabric Loader `0.19.3+`
- Fabric API

## Optional

- LuckPerms for permission management.
- Mod Menu for the optional local config editor.
- Lodestone Warps installed on the client for the enhanced custom UI.

## Server-Only Setup

1. Install Fabric Loader on the server.
2. Add Fabric API.
3. Add `lodestone-warps-<version>.jar`.
4. Start the server once to generate the config.
5. Edit `config/lodestone_warp_and_tp/lodestone_teleport.json` if needed.

Vanilla players can join without installing the mod.

## Client Mod Setup

Players may also install Lodestone Warps client-side.

This enables the custom mod UI with:

- table layout
- search
- pagination
- XP/item cost icons
- local favorites
- configurable columns
- edit buttons

Players without the client mod continue using the vanilla Dialog UI.
