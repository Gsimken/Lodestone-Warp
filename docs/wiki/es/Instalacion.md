# Instalación

[English](../Installation.md) | [Español](Instalacion.md)

**Última actualización:** 2026-07-13

## Requisitos del Servidor

- Minecraft `26.2`
- Java `25+`
- Fabric Loader `0.19.3+`
- Fabric API

## Opcional

- LuckPerms para administrar permisos.
- Mod Menu para el editor opcional de config local.
- Lodestone Warps instalado en el cliente para activar la UI de mod.

## Instalación Solo Servidor

1. Instala Fabric Loader en el servidor.
2. Agrega Fabric API.
3. Agrega `lodestone-warps-<version>.jar`.
4. Inicia el servidor una vez para generar la configuración.
5. Edita `config/lodestone_warp_and_tp/lodestone_teleport.json` si lo necesitas.

Los jugadores vanilla pueden entrar sin instalar el mod.

## Instalación en Cliente

Los jugadores también pueden instalar Lodestone Warps en el cliente.

Esto activa la UI de mod con:

- tabla de destinos
- búsqueda
- paginación
- iconos de costo por XP/item
- favoritos locales
- columnas configurables
- botones de edición

Los jugadores sin el mod siguen usando la UI vanilla.
